== Tutorial: Accelerometer
	
We will build an application which will change its background color
if it is shuffled.
Create a new Android project called
_de.vogella.android.sensor_
with
an
_activity_
called
_SensorTestActivity_.
	
Change your layout file
to the following code.
	
[source,xml]
----
include::res/accelerometer/main.xml[]
----	
	
Change your
_activity_
class
to the following code.
	
[source,java]
----
include::res/accelerometer/SensorTestActivity.java[]
----
