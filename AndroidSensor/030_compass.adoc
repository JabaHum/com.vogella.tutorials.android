== Tutorial: Building a compass
	
Create a new Android project called
_de.vogella.android.sensor.compass_
with
an
_activity_
called
_MainActivity_.
	
Create the following custom
`View`
class.
	
[source,java]
----
include::res/compass/MyCompassView.java[]
----	
	
Change the coding of your
_activity_.
	
[source,java]
----
include::res/compass/MainActivity.java[]
----
